ENGLISH INSTRUCTION

1, 2, 3, 4, 5: dispensers 
Q, W, E, R, T, Y: glasses
A, S, D, F: bottles
Z, X, C, V, B: filled bottles
Space: straw

drag object to move
click objects with ctrl to remove

click dispenser to change liquid
click dispenser with ctrl to set first liquid










POLSKA INSTRUKCJA

klawisze 1 - 5: dozownik leje płyny (0 - 4: dozowniki boczne, 5: dozownik tęcza)
klawisze q - y: spawn szklanki
klawisze a - f: spawn butelki
klawisze z - v: spawn pełnej butelki (piwo, cola, wódka, whisky, wino)
spacja: spawn słomki

przesuwanie obiektów za pomocą przeciągania LPM (nie działa na dispenserach)
usuwanie obiektów za pomocą LPM + Ctrl

zmiana aktualnie nalewanego płynu przez dispenser z pomocą kliknięcia go LPM (piwo, whisky, wino, wódk, cola, sok malinowy, sok bananowy)
zmiana aktualnie nalewanego płynu przez dispensera na pierwszy z pomocą kliknięcia go LPM + Ctrl
